Angular TouchSelect Demo

TouchSelect is an angular directive that provides an alternative to a traditional select element for interactive web apps. It adds a pop-up options dialog to your element which can be used to select the value of a scope variable. Just click and drag or touch and drag on touch-enabled devices to select an option.

The directive will react to changes to the disabled parameter so it can be set dynamically, as in this example which only permits valid race outcomes to be selected. 
$scope.places = ['-', '-', '-'];
$scope.placeChoices = [
  {value: '1st'}, {value: '2nd'}, {value: '3rd'}, {value: '-'}
];
$scope.$watch('places', function() {
  for (var i = 0; i < $scope.places.length; i++) {
    var alreadyUsed = false;
    for (var j = 0; j < $scope.places.length; j++) {
      if ($scope.places[j] === $scope.placeChoices[i].value) {
        alreadyUsed = true;
      }
    }
    $scope.placeChoices[i].disabled = alreadyUsed;
  }
}, true);
